# Apistration

API Manager for [API Entreprise](https://entreprise.api.gouv.fr) and [API Particulier](https://particulier.api.gouv.fr) — French government APIs providing certified data about companies and individuals.

Built and maintained by [DINUM](https://www.numerique.gouv.fr/) (Direction Interministérielle du Numérique).

## Structure

The application lives in the [`siade/`](siade/) directory (Système d'Information des API de l'État).

## Getting started

See [`siade/README.md`](siade/README.md) for requirements, installation, and development instructions.
